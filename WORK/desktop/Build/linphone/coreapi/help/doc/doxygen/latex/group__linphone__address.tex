\subsection{Detailed Description}
S\+IP address parser A\+PI. 

This api is useful for manipulating S\+IP addresses (\textquotesingle{}from\textquotesingle{} or \textquotesingle{}to\textquotesingle{} headers).

\subsubsection{Linphone\+Address}
{\footnotesize\ttfamily typedef struct Sal\+Address \textbf{ Linphone\+Address}}



Object that represents a S\+IP address. 

The Linphone\+Address is an opaque object to represents S\+IP addresses, ie the content of S\+IP\textquotesingle{}s \textquotesingle{}from\textquotesingle{} and \textquotesingle{}to\textquotesingle{} headers. A S\+IP address is made of display name, username, domain name, port, and various uri headers (such as tags). It looks like \textquotesingle{}Alice $<$sip\+:{\tt alice@example.\+net}$>$\textquotesingle{}. The Linphone\+Address has methods to extract and manipulate all parts of the address. When some part of the address (for example the username) is empty, the accessor methods return N\+U\+LL.
