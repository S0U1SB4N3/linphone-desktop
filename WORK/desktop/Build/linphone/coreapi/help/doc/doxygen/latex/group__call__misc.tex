\subsection{Detailed Description}
Obtaining information about a running call\+: sound volumes, quality indicators. 

When a call is running, it is possible to retrieve in real time current measured volumes and quality indicator.

\subsubsection{Linphone\+Call\+Stats}
{\footnotesize\ttfamily typedef struct \+\_\+\+Linphone\+Call\+Stats \textbf{ Linphone\+Call\+Stats}}



The Linphone\+Call\+Stats objects carries various statistic informations regarding quality of audio or video streams. 

To receive these informations periodically and as soon as they are computed, the application is invited to place a \doxyref{Linphone\+Core\+Call\+Stats\+Updated\+Cb}{p.}{group__initializing_gae50aa77005e69df0137977b45bc9e832} callback in the Linphone\+Core\+V\+Table structure it passes for instanciating the Linphone\+Core object (see \doxyref{linphone\+\_\+core\+\_\+new()}{p.}{group__initializing_ga9601ae3c81a1bbf110e84a1a484f3fee} ).

At any time, the application can access last computed statistics using linphone\+\_\+call\+\_\+get\+\_\+audio\+\_\+stats() or linphone\+\_\+call\+\_\+get\+\_\+video\+\_\+stats().

\subsubsection{linphone\+\_\+call\+\_\+get\+\_\+current\+\_\+quality()}
{\footnotesize\ttfamily float linphone\+\_\+call\+\_\+get\+\_\+current\+\_\+quality (\begin{DoxyParamCaption}\item[{\textbf{ Linphone\+Call} $\ast$}]{call }\end{DoxyParamCaption})}



Obtain real-\/time quality rating of the call. 

Based on local R\+TP statistics and R\+T\+CP feedback, a quality rating is computed and updated during all the duration of the call. This function returns its value at the time of the function call. It is expected that the rating is updated at least every 5 seconds or so. The rating is a floating point number comprised between 0 and 5.

4-\/5 = good quality ~\newline
 3-\/4 = average quality ~\newline
 2-\/3 = poor quality ~\newline
 1-\/2 = very poor quality ~\newline
 0-\/1 = can\textquotesingle{}t be worse, mostly unusable ~\newline


\begin{DoxyReturn}{Returns}
The function returns -\/1 if no quality measurement is available, for example if no active audio stream exist. Otherwise it returns the quality rating. 
\end{DoxyReturn}
