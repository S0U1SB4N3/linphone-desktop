\subsection{Detailed Description}
Chat room and Messaging. 

{\bfseries  Exchanging text messages} ~\newline
 Messages are sent using \doxyref{Linphone\+Chat\+Room}{p.}{group__chatroom_ga37902f3fc715b7f6451c241d0efd0299} object. First step is to create a \doxyref{chat room }{p.}{group__chatroom_gae91f97feaeea9a7dedab24faf3fbc3c2} from a peer sip uri. 
\begin{DoxyCode}
LinphoneChatRoom* chat\_room = linphone_core_get_chat_room(lc,\textcolor{stringliteral}{"sip:joe@sip.linphone.org"});
\end{DoxyCode}


~\newline
Once created, messages are sent using function \doxyref{linphone\+\_\+chat\+\_\+room\+\_\+send\+\_\+message()}{p.}{group__chatroom_gad22a10b819a4a33f6dc12d88b64e5537}. 
\begin{DoxyCode}
linphone_chat_room_send_message(chat\_room,\textcolor{stringliteral}{"Hello world"}); \textcolor{comment}{/*sending message*/}
\end{DoxyCode}
 ~\newline
Incoming message are received from call back \doxyref{Linphone\+Core\+V\+Table.\+text\+\_\+received}{p.}{struct__LinphoneCoreVTable_aa8c03429942bf96b1ad5420d406114c8} 
\begin{DoxyCode}
\textcolor{keywordtype}{void} text\_received(LinphoneCore *lc, LinphoneChatRoom *room, \textcolor{keyword}{const} 
      LinphoneAddress *from, \textcolor{keyword}{const} \textcolor{keywordtype}{char} *message) \{
        printf(\textcolor{stringliteral}{" Message [%s] received from [%s] \(\backslash\)n"},message,
      linphone_address_as_string (from));
\}
\end{DoxyCode}
 ~\newline
 A complete tutorial can be found at \+: \doxyref{Chat room tutorial}{p.}{group__chatroom__tuto}
