\subsection{Detailed Description}
Managing proxies. 

User registration is controled by \doxyref{Linphone\+Proxy\+Config}{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} settings.~\newline
 Each \doxyref{Linphone\+Proxy\+Config}{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} object can be configured with registration informations like \doxyref{proxy address }{p.}{group__proxies_ga6e896345be76aaf6399ab4ed4532b2a4} , \doxyref{user id }{p.}{group__proxies_ga978c00a218b907a2f98fc59f5d217805}, \doxyref{refresh period }{p.}{ga2929b27c5f57444aa2656e27bbedd1b0}, and so on. ~\newline
 A created proxy config using \doxyref{linphone\+\_\+proxy\+\_\+config\+\_\+new()}{p.}{group__proxies_gac435e6efb12fedf8a016745c88ee09a4}, once configured, must be added to \doxyref{Linphone\+Core}{p.}{group__initializing_gaa5cf635b82dd338e7ee2dd3599d05f0a} using function \doxyref{linphone\+\_\+core\+\_\+add\+\_\+proxy\+\_\+config()}{p.}{group__proxies_gafe9c40e742c5b047fc9a6825da97d075}. ~\newline
 It is recommended to set a default \doxyref{proxy config }{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} using function \doxyref{linphone\+\_\+core\+\_\+set\+\_\+default\+\_\+proxy()}{p.}{group__proxies_ga2082f2c37ffbf5d46e8d20ae8bccc2a4}. Once done, if \doxyref{a proxy config }{p.}{group__proxies_ga1879a0d59e068457b688138da97f2d0b} has been configured with attribute \doxyref{enable register }{p.}{group__proxies_gaea39579996645d4bc5e62e68ab097f17} , next call to \doxyref{linphone\+\_\+core\+\_\+iterate()}{p.}{group__initializing_ga7bdac8ac386483fc4e0876e4c5d13755} triggers a S\+IP register. ~\newline
 Registration status is reported by Linphone\+Core\+Registration\+State\+Changed\+Cb. ~\newline

\subsubsection{Linphone\+Proxy\+Config}
{\footnotesize\ttfamily typedef struct \+\_\+\+Linphone\+Proxy\+Config \textbf{ Linphone\+Proxy\+Config}}



The Linphone\+Proxy\+Config object represents a proxy configuration to be used by the Linphone\+Core object. 

Its fields must not be used directly in favour of the accessors methods. Once created and filled properly the Linphone\+Proxy\+Config can be given to Linphone\+Core with \doxyref{linphone\+\_\+core\+\_\+add\+\_\+proxy\+\_\+config()}{p.}{group__proxies_gafe9c40e742c5b047fc9a6825da97d075}. This will automatically triggers the registration, if enabled.

The proxy configuration are persistent to restarts because they are saved in the configuration file. As a consequence, after \doxyref{linphone\+\_\+core\+\_\+new()}{p.}{group__initializing_ga9601ae3c81a1bbf110e84a1a484f3fee} there might already be a list of configured proxy that can be examined with \doxyref{linphone\+\_\+core\+\_\+get\+\_\+proxy\+\_\+config\+\_\+list()}{p.}{group__proxies_ga23b6e78aae85500af63955a2be36eaff}.

The default proxy (see \doxyref{linphone\+\_\+core\+\_\+set\+\_\+default\+\_\+proxy()}{p.}{group__proxies_ga2082f2c37ffbf5d46e8d20ae8bccc2a4} ) is the one of the list that is used by default for calls.
