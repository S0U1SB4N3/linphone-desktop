\subsection{Detailed Description}
An object representing a bi-\/directional R\+TP session. It holds sockets, jitter buffer, various counters (timestamp, sequence numbers...) Applications S\+H\+O\+U\+LD N\+OT try to read things within the Rtp\+Session object but use instead its public A\+PI (the rtp\+\_\+session\+\_\+$\ast$ methods) where Rtp\+Session is used as a pointer. \doxyref{rtp\+\_\+session\+\_\+new()}{p.}{rtpsession_8h_a0fa068a78f69d25603baf8801cd352b7} allocates and initialize a Rtp\+Session.
